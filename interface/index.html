Demo Interface en utilisant l'API Netatmo

Technologies utilisées : Bootstrap,AngularJS,NodeJS,ExpressJS

Autre API utilisée: Opencagedata

Lors de la recherche d'une ville, les coordonnées sont retournées par l'API Opencagedata.

Une requête est alors faites avec ces coordonnées vers l'API Netatmo qui renvoie alors tous les devices présent dans cette zones.

A l'aide d'une boucle je selectionne uniquement les modules mesurant la température et fais la moyenne de tous les résultats obtenus.
